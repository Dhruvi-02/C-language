# This folder contains conditional statement concepts related codes of C Programming Language

- In C language conditional statements are used to make decisions and control the flow of a program based on given conditions.
- They allow the program to execute certain parts of code only when specific conditions are true.

# Topics included
- if => Executes code if a condition is true
- else-if => Chooses between two possible actions
- nested else-if => One if statement inside another, for complex conditions
- if-elseif => Checks multiple conditions in sequence
